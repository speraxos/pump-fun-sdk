## How to Add Your Project

1. Fork this repo
2. Add your project to the appropriate table above
3. Open a PR with the title `[adopters] Add <your project name>`

Please include:
- **Project name** — your project or company name
- **Description** — one sentence about what you do with pump-fun-sdk
- **Link** — website, GitHub repo, or relevant URL

We accept all legitimate use cases. Open-source and commercial projects welcome.
